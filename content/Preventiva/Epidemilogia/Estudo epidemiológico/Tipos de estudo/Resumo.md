### <font color="#245bdb">ECR x Coorte prospectivo x Caso controle</font>

|Característica|Ensaio Clínico Randomizado (ECR)|Coorte Prospectivo|Caso-Controle|
|---|---|---|---|
|**Quem decide a exposição?**|O pesquisador ALOCA os participantes|O participante JÁ ESTÁ exposto naturalmente|O participante JÁ TEM ou não a doença|
|**Ponto de partida**|Randomização → grupos|Exposição (já existente) → seguimento|Desfecho (doença) → investigação retrospectiva|
|**Direção temporal**|Prospectivo: exposição → desfecho|Prospectivo: exposição → desfecho|Retrospectivo: desfecho → exposição|
|**Palavra-chave no enunciado**|"alocados", "randomizados", "designados"|"acompanhados", "seguidos", "expostos vs não expostos"|"casos vs controles", "pacientes com doença X"|
|**Medida de associação**|Risco Relativo (RR)|Risco Relativo (RR)|Odds Ratio (OR)|
|**Estabelece causalidade?**|SIM (padrão-ouro)|Sugere, mas não prova|Não (apenas associação)|

- **ECR:** O pesquisador **INTERVÉM** — ele decide quem recebe o tratamento/rastreamento
- **Coorte:** O pesquisador **OBSERVA** — ele apenas acompanha pessoas que já estão expostas por escolha própria ou circunstâncias

![[Pasted image 20260407211027.png]]

Esta figura ilustra a diferença fundamental: no ECR (painel superior), a randomização garante distribuição igual de fatores prognósticos entre os grupos. Nos estudos observacionais (painel inferior), fatores conhecidos e desconhecidos influenciam quem recebe cada tratamento, criando viés.

## Direcionamento

| Tipo de Estudo                 | Melhor Para                                                                                                   | Vantagens                                                          | Limitações                                                  |
| ------------------------------ | ------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------ | ----------------------------------------------------------- |
| **Relato de caso**             | Doença nova/desconhecida; apresentação clínica rara ou atípica                                                | Gera hipóteses; primeiro alerta para novas condições               | Não estabelece causalidade; sem grupo comparação            |
| **Série de casos**             | Descrever padrões de doença rara; <br>identificar características comuns                                      | Gera hipóteses; útil em surtos iniciais                            | Sem grupo controle; não calcula incidência                  |
| **Transversal**                | Calcular prevalência; planejamento em saúde pública                                                           | Rápido; barato; usa dados existentes                               | Não estabelece temporalidade; não calcula incidência        |
| **Ecológico**                  | Tendências populacionais; comparar regiões/países; gerar hipóteses                                            | Rápido; barato; usa dados agregados públicos                       | Falácia ecológica; não analisa indivíduos                   |
| **Caso-controle**              | Doença rara; <br>surtos/epidemias; <br>múltiplos fatores de risco; <br>efeitos adversos raros de medicamentos | Eficiente para doenças raras; rápido; avalia múltiplas exposições  | Viés de memória; não calcula incidência; só mede Odds Ratio |
| **Coorte**                     | Exposição rara; <br>calcular incidência; <br>múltiplos desfechos de uma exposição; <br>doença comum           | Estabelece temporalidade; calcula risco relativo e incidência      | Caro; demorado; perdas de seguimento                        |
| **Ensaio clínico randomizado** | Testar eficácia de medicamentos/tratamentos; estabelecer causalidade                                          | Padrão-ouro; randomização elimina vieses; maior nível de evidência | Caro; questões éticas; pode não refletir prática real       |

## Vieses
| Tipo de Estudo                      | Principais Vieses                                                          | Descrição do Viés                                                                                 | Como Minimizar                                                                                |
| ----------------------------------- | -------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------- |
| **Relato de caso / Série de casos** | Viés de seleção; Viés de publicação                                        | Casos selecionados por serem interessantes/atípicos; casos positivos são mais publicados          | Descrever critérios de inclusão; reconhecer limitações                                        |
| **Transversal**                     | Viés de prevalência (sobrevivência); Viés de informação                    | Captura apenas casos prevalentes (sobreviventes); depende de autorrelato                          | Usar medidas objetivas; validar questionários                                                 |
| **Ecológico**                       | Falácia ecológica; Viés de agregação                                       | Associações em nível populacional não se aplicam a indivíduos; dados agregados mascaram variações | Reconhecer limitação; usar para gerar hipóteses apenas                                        |
| **Caso-controle**                   | Viés de memória (recall); Viés de seleção de controles; Viés de informação | Casos lembram mais exposições que controles; controles inadequados distorcem resultados           | Usar registros objetivos; selecionar controles da mesma população-base; cegar entrevistadores |
| **Coorte**                          | Viés de perda de seguimento; Viés do trabalhador sadio; Viés de informação | Perdas diferenciais entre grupos; populações ocupacionais são mais saudáveis                      | Minimizar perdas; análise de sensibilidade; cegar avaliadores de desfecho                     |
| **Ensaio clínico randomizado**      | Viés de performance; Viés de detecção; Viés de atrito                      | Tratamento diferencial se não cegado; avaliação diferencial do desfecho; perdas diferenciais      | Duplo-cegamento; análise por intenção de tratar; protocolo rigoroso                           |
