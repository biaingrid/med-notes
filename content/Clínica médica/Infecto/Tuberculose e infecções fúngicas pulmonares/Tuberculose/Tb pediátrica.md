### Sistema de pontuação do Ministério da Saúde do Brasil
Uma ferramenta validada para auxiliar no diagnóstico clínico quando a confirmação bacteriológica não é possível.O sistema brasileiro avalia cinco critérios principais, atribuindo pontos para cada um: 

| Critério                                                                           | Pontuação  |
| :--------------------------------------------------------------------------------- | :--------: |
| **Quadro clínico-radiológico**                                                     |            |
| - Adenomegalia hilar ou padrão miliar                                              | +15 pontos |
| - Condensação ou infiltrado inalterado ≥2 semanas                                  | +15 pontos |
| - Condensação ou infiltrado com melhora após antibiótico para germes comuns        | +5 pontos  |
| - Radiografia normal                                                               | -5 pontos  |
| <mark style="background:rgba(240, 200, 0, 0.2)">**Contato com adulto tuberculoso** |  </mark>   |
| - Próximo, nos últimos 2 anos                                                      | +10 pontos |
| - Ocasional ou negativo                                                            |  0 pontos  |
| **Prova tuberculínica (PT/PPD)**                                                   |            |
| - ≥10 mm (ou ≥5 mm em imunodeprimidos)                                             | +10 pontos |
| - 5-9 mm                                                                           | +5 pontos  |
| - 5 mm                                                                             |  0 pontos  |
| **Estado nutricional**                                                             |            |
| - Desnutrição grave (peso 10º percentil)                                           | +5 pontos  |
| **Sintomas**                                                                       |            |
| - Febre, tosse, adinamia, sudorese ≥2 semanas                                      | +15 pontos |
- **≥40 pontos**: Diagnóstico muito provável → iniciar tratamento
- **30-35 pontos**: Diagnóstico possível → indicado iniciar tratamento a critério médico
- **<30 pontos**: Diagnóstico pouco provável → continuar investigação
