A AAST-OIS (American Association for the Surgery of Trauma - Organ Injury Scale) é o <mark style="background:rgba(240, 200, 0, 0.2)">sistema de classificação de lesões de órgãos</mark>mais aceito mundialmente para trauma abdominal contuso. 
Foi desenvolvida pela AAST em 1989, revisada em 1994 e atualizada novamente em 2018.

### Finalidade
| Finalidade                              | Descrição                                                                       |
| --------------------------------------- | ------------------------------------------------------------------------------- |
| **Estratificação de gravidade**         | Classifica lesões de órgãos sólidos em graus I-V <br>(ou VI para alguns órgãos) |
| **Guiar decisões de manejo**            | Ajuda a decidir entre MNO, embolização ou cirurgia                              |
| **Pesquisa clínica**                    | Permite comparação padronizada de resultados entre centros                      |
| **Faturamento e codificação**           | Utilizada para fins administrativos hospitalares                                |
| **Credenciamento de centros de trauma** | Requisito para certificação de centros de trauma nos EUA                        |
| **Predição de desfechos**               | Correlaciona-se com mortalidade, taxa operatória e tempo de internação          |

### Classificação
A versão de 2018 classifica as lesões em 3 critérios

| Critério                    | Descrição                          |
| --------------------------- | ---------------------------------- |
| **Imagem (Imaging)**        | Achados na TC com contraste        |
| **Operatório (Operative)**  | Achados durante cirurgia           |
| **Patológico (Pathologic)** | Achados em peça cirúrgica/autópsia |

#### Órgãos classificados pela AAST-OIS
| Categoria                     | Órgãos                                                         |
| ----------------------------- | -------------------------------------------------------------- |
| **Órgãos sólidos abdominais** | Baço, Fígado, Rim                                              |
| **Órgãos ocos**               | Estômago, Intestino delgado, Cólon, Reto                       |
| **Outros**                    | Pâncreas, Duodeno, Bexiga, Ureter, Diafragma, Vasos abdominais |

#### Escala de graus
|  Grau   |       Gravidade       |                                  Descrição Geral                                   |
| :-----: | :-------------------: | :--------------------------------------------------------------------------------: |
|  **I**  |        Mínima         |                  Hematoma subcapsular <10% ou <br>laceração <1 cm                  |
| **II**  |         Leve          |                Hematoma subcapsular 10-50% ou <br>laceração 1-3 cm                 |
| **III** |       Moderada        | Hematoma subcapsular >50% ou <br>laceração >3 cm OU <br>múltiplas lesões grau I-II |
| **IV**  |         Grave         |      Laceração envolvendo >25% do parênquima ou <br>lesão vascular segmentar       |
|  **V**  |        Crítica        |              Destruição completa do órgão ou <br>lesão vascular hilar              |
| **VI**  | Incompatível com vida |                          Avulsão completa (apenas fígado)                          |
<mark style="background:rgba(240, 200, 0, 0.2)">Se houver múltiplas lesões de grau I ou II no mesmo órgão, o grau é automaticamente elevado para grau III.</mark>

#### Achados vasculares
A atualização de 2018 incorporou <mark style="background:rgba(240, 200, 0, 0.2)">achados vasculares na TC</mark> que automaticamente elevam o grau:

| Achado Vascular                               | Significado            | Impacto no Grau       |
| --------------------------------------------- | ---------------------- | --------------------- |
| **Extravasamento ativo de contraste (blush)** | Sangramento ativo      | Eleva para ≥ grau III |
| **Pseudoaneurisma**                           | Lesão arterial contida | Eleva para ≥ grau III |
| **Fístula arteriovenosa**                     | Comunicação anormal AV | Eleva para ≥ grau III |
