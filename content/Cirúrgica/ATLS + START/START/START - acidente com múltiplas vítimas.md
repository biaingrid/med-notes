### Definição e classificação
Sistema de Triagem START (Simple Triage and Rapid Treatment)
Em incidentes com múltiplas vítimas, como tiroteios em massa, utiliza-se o sistema START modificado, que classifica pacientes em **4 categorias** baseadas em avaliação rápida:

![[Pasted image 20260404182527.png]]

**Categorias de prioridade**
1.<mark style="background:rgba(244, 81, 81, 0.55)">IMEDIATO (Vermelho)</mark> - Prioridade máxima para avaliação e tratamento
    - Requer intervenção salvadora de vida
    - Tem chance razoável de sobrevivência com tratamento imediato
    - Exemplos: hemorragia ativa controlável, obstrução de via aérea, pneumotórax hipertensivo

2.<mark style="background:rgba(240, 200, 0, 0.2)">ATRASADO (Amarelo)</mark> - Segunda prioridade
    - Lesões graves, mas pode aguardar horas para tratamento
    - Sinais vitais estáveis

3. **<mark style="background:rgba(3, 135, 102, 0.2)">MÍNIMO (Verde)</mark>** - Terceira prioridade
    - "Walking wounded" - feridos que conseguem andar
    - Lesões menores, ambulatoriais

4. <mark style="background:rgba(92, 92, 92, 0.2)">EXPECTANTE (Preto)</mark> - Sem prioridade
    - Mortos ou lesões incompatíveis com sobrevivência
    - Recursos insuficientes para salvar

### Prioridade para avaliação posterior
O paciente com prioridade para avaliação posterior seria <mark style="background:#fff88f">aquele classificado como IMEDIATO (vermelho)</mark> - ou seja, aquele que:
- Apresenta condição potencialmente fatal
- Necessita intervenção urgente (controle de hemorragia, via aérea, descompressão torácica)
- Tem probabilidade razoável de sobrevivência se tratado rapidamente
