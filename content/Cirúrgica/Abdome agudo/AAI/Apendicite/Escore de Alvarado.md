O Escore de Alvarado, desenvolvido em 1986, é um sistema de pontuação clínica bem conhecido, frequentemente utilizado por médicos emergencistas para <mark style="background:rgba(240, 200, 0, 0.2)">auxiliar na detecção de apendicite e determinar a necessidade de tomografia computadorizada</mark>. É um sistema de pontuação clínica que varia de **1 a 10 pontos**, baseado em achados clínicos e laboratoriais simples. Os componentes incluem:

| Componente                                  | Pontos |
| ------------------------------------------- | ------ |
| **Sintomas**                                |        |
| Migração da dor para fossa ilíaca direita   | 1      |
| Anorexia                                    | 1      |
| Náusea/vômito                               | 1      |
| **Sinais**                                  |        |
| Sensibilidade em quadrante inferior direito | 2      |
| Dor à descompressão (sinal de Blumberg)     | 1      |
| Febre (>37,5°C)                             | 1      |
| **Laboratório**                             |        |
| Leucocitose (>10.000/μL)                    | 2      |
| Desvio à esquerda (>75% neutrófilos)        | 1      |
- **1-4 pontos:** Apendicite improvável
- **5-6 pontos:** Compatível com apendicite aguda
- **7-8 pontos:** Apendicite provável
- **9-10 pontos:** Apendicite muito provável
