### Definição
O **Escore de Caprini** é um modelo de avaliação de risco para **tromboembolismo venoso (TEV)** em pacientes cirúrgicos, incorporando aproximadamente 40 fatores de risco individuais em um modelo ponderado para criar uma pontuação agregada.

### Pontuação
O escore atribui pontos conforme os fatores de risco presentes:

|    Pontos    |                                                                                                                                 Fatores de Risco                                                                                                                                  |
| :----------: | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------: |
| **1 ponto**  | Idade 41-60 anos, cirurgia menor, IMC >25, pernas edemaciadas, varizes, gravidez/pós-parto, história de abortos inexplicados (>3), uso de contraceptivos/TRH, sepse (1 mês), pneumonia (1 mês), função pulmonar anormal, IAM agudo, ICC (1 mês), DII, paciente clínico em repouso |
| **2 pontos** |                                                                Idade 61-74 anos, cirurgia aberta maior (>45 min), cirurgia laparoscópica (>45 min), malignidade, confinado ao leito (>72h), acesso venoso central                                                                 |
| **3 pontos** |                                    Idade ≥75 anos, história de TVP/TEP, história familiar de trombose, Fator V Leiden, Protrombina 20210A, anticoagulante lúpico, anticorpos anticardiolipina, homocisteína elevada, TIH, outras trombofilias                                     |
| **5 pontos** |                                                                                          AVC (1 mês), artroplastia eletiva, fratura de quadril/pelve/perna, lesão medular aguda (1 mês)                                                                                           |
*Se houver mais de um fator de risco na mesma pontuação, adiciona-se a pontuação. Ex: paciente com 54 anos, com cirurgia >2h e presença de malignidade: 1+2+2.*

| Pontuação | Categoria de Risco | Risco Basal de TEV (sem profilaxia) |
| :-------: | :----------------: | :---------------------------------: |
|    0-1    |    Muito baixo     |                0,5%                 |
|     2     |       Baixo        |                ~1,5%                |
|    3-4    |      Moderado      |                ~3,0%                |
|    ≥5     |        Alto        |                ~6,0%                |

### Profilaxia

| Escore Caprini |           Profilaxia Recomendada           |
| :------------: | :----------------------------------------: |
|       0        |         Deambulação precoce apenas         |
|      1-2       | Profilaxia mecânica (CPI) OU farmacológica |
|      3-4       |    Profilaxia farmacológica ± mecânica     |
|       ≥5       |    Profilaxia farmacológica + mecânica     |
